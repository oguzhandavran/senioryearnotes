<span style="color:rgb(0, 176, 240)"><br></span>## 🧠 2 — Konu Hâkimiyeti (her birini "yapabiliyorum" de)

- [x] **Mekanik → TF:** FBD çiz → Newton/Lagrange → Laplace → Cramer → $G(s)$
- [x] **İki kütle bağlı sistem:** $\frac{X_2}{F}=\frac{Z}{\Delta_1\Delta_2-Z^2}$ kurabiliyorum
- [x] **Elektrik:** empedans ($Ls$, $1/Cs$), Mesh KVL, Op-amp ($-Z_f/Z_{in}$)
- [x] **Durum uzayı:** eşlik kanonik form $A,B,C,D$ yazabiliyorum
- [x] **SS → TF:** $G(s)=C(sI-A)^{-1}B+D$, 2×2 ters alabiliyorum
- [ ] **Doğrusallaştırma:** denge noktası ($\dot{x}=0$) → Jacobian → $\delta\dot{x}=A\delta x$
- [ ] **2. derece parametreler:** $\omega_n=\sqrt{k/m}$, $\zeta=b/(2\sqrt{mk})$
- [ ] **★ KYE & Kompansasyon (Nise Bölüm 9):** PD/PI/PID, Lead/Lag — hoca bizzat işaret etti!

## ⚠️ 3 — Son Tuzak Taraması

- [ ] Lagrange'da **Rayleigh sönüm ($D$) terimi** unutma → sönüm kaybolmasın
- [ ] Dişli: $\Theta_1/T_1$ mi $\Theta_2/T_2$ mi sorulmuş, **iki kez oku**
- [ ] Cramer 2×2 determinantını **tekrar kontrol et**
- [ ] $(sI-A)^{-1}$: adjoint'te **köşegen elemanlar yer değiştirir**
- [ ] Birim dönüşümü: N/mm→×1000, rpm→×$2\pi/60$, mH→×0.001, μF→×$10^{-6}$
- [ ] Durum uzayında fiziksel sistemde genelde $D=0$
